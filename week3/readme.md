### Backend Development (Week 3)

1. Create git repository  
   git init

2. Add `.gitignore` file  
   Used to ignore unnecessary files

3. Generate `package.json`  
   npm init -y

4. Install Express framework  
   npm install express

5. Create Express server  
   Create `server.js` and start server using Express

6. Create API folder  
   Inside API folder, create User API and Product API files

7. Design API routes  
   Implement GET, POST, PUT and DELETE routes

8. Handle request and response  
   Read request data and send response using Express

9. Store data temporarily  
   Use in-memory arrays instead of database

10. Test APIs  
    Test all APIs using `req.http`
